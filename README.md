# 🎵 Spotify Clone

A simple music streaming web app built with **HTML, CSS, and JavaScript**. It allows users to **play, pause, and search for songs** using an external API.

## 🌟 Features
✔️ Play & Pause Songs 🎶  
✔️ Search for Music 🔍  
✔️ Responsive & Modern UI 🎨
